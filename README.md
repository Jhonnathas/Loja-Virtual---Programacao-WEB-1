# Loja-Virtual---Programacao-WEB-1
Desenvolver uma plataforma de e-commerce para uma editora de materiais bibliográficos da área de Inteligência Artificial. A plataforma deve ser construída para permitir a venda de livros físicos e digitais, de forma que seja de fácil gerenciamento por pessoas sem conhecimento técnico, aceite pagamentos com cartão e PIX.
